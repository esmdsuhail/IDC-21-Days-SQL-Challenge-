# IDC-21-Days-SQL-Challenge-
21-day SQL program covering fundamentals to advanced: queries, joins, subqueries, CTEs, window functions, optimization, and data modeling. Includes daily topics with practical datasets and exercises to build strong skills for Data Analysis, Data Engineering, Data Science, and Software Development by [Indian Data Club](https://www.indiandataclub.com/)
